Adagrams

Adagrams is a game that I designed in which a player is given a random set of letters and must make an anagram out of those letters. Each word is then scored.

Task List API

In this project, I created a web API to organize tasks. This API allows users to create, read, update, and delete tasks. It also utilizes the Slack web API to notify via slack when a task is marked complete.

Swap Meet

This project organized a virtual swap meet. Theoretically, each vendor would register online and submit a list of their inventory. This project represents the back end logic of a swap meet application, including a rating system for all items in one's inventory and sorting all inventory into three categories: clothing, decor, and electronics.
